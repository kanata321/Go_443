# Go_443

A checker for HTTPS Certificates

## Description

Go_443 is CLI tool that check HTTPS Certificates.

## Features

- Easily validate
- Faster Check (using 'goroutine')
- Output infomation of certificates as JSON

For more information, see `go443(.exe, .app) --help`.

## Requirement

- Go1.9
